📸 ATTEND PRO | Next-Gen Biometric Attendance System
========================================================================

An advanced classroom management platform revolutionizing traditional 
roll-calls with cutting-edge computer vision and voice biometrics. 
ATTEND PRO eliminates manual data entry, proxy attendance, and 
administrative headaches by automating the entire roster workflow.

🌐 Live Landing Page: https://attendpro-landing-page.vercel.app/
🚀 Core Application App: https://attendpro-main.streamlit.app/

------------------------------------------------------------------------
✨ KEY FEATURES
------------------------------------------------------------------------

* AI Face Analysis:
  Instantly scans an entire classroom from a single group photo. 
  Utilizes advanced deep learning neural networks to detect and verify 
  every student's face simultaneously.

* Sequential Voice ID:
  A futuristic, hands-free roll-call system. Students call out 
  "Present" sequentially, and our real-time audio-AI matches their 
  unique vocal signatures against high-fidelity voice embeddings.

* QR-Driven Roster:
  Instant, hassle-free student enrollment. Generates secure, dynamic 
  course-specific QR codes for student onboarding in seconds.

* Unified Teacher Dashboard:
  Complete operational visibility. Teachers can manage subjects, 
  generate attendance logs, track long-term attendance trends, view 
  AI confidence scores, and download comprehensive CSV reports.

* Secure Student Portal:
  Complete personal tracking where students can view their attendance 
  logs, calculate percentages across registered subjects, and complete 
  initial biometric profile setup.

------------------------------------------------------------------------
🛠️ ADVANCED TECH STACK
------------------------------------------------------------------------

ATTEND PRO is engineered using a robust hybrid architecture combining 
reactive frontends with distributed cloud microservices:

* Frontend Layer: Reactive UI components built using Streamlit backed 
  by an interactive, optimized Flask landing layer structured with 
  semantic HTML5 and customized responsive CSS3 variables.

* Computer Vision Framework: High-fidelity facial detection, alignment, 
  and encoding generation powered by Face-Recognition API and Dlib.

* Audio Biometrics Infrastructure: Secure vocal feature extraction and 
  speaker diarization utilizing Resemblyzer neural networks and 
  Librosa signal processing.

* Database & Storage Engine: Distributed PostgreSQL database layer with 
  real-time replication, cloud storage buckets, and secure row-level 
  security (RLS) managed via Supabase Cloud.

------------------------------------------------------------------------
🗺️ SYSTEM WORKFLOW
------------------------------------------------------------------------

👨‍🏫 The Teacher's Journey:
  1. Secure Access: Centralized secure management portal login.
  2. Classroom Feed: Streamlined viewing of active courses and logs.
  3. Course Setup: Single-click creation of a new subject tracking code.
  4. Biometric Roll-Call: Run FaceID image scan or sequential Voice ID.
  5. Data Export: Review analytics and export verified .csv reports.

🎓 The Student's Journey:
  1. Onboarding: Scan the course QR code for instant roster entry.
  2. Biometric Calibration: One-time profile setup for baseline face 
     embeddings and audio signatures.
  3. Performance Tracking: View individual logs and target attendance 
     benchmarks via the student dashboard.
